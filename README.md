# Documentación de Proyecto VRChat y Dispositivos Hápticos

Descripción
Este repositorio contiene la documentación del análisis e investigación sobre el uso de la plataforma VRChat junto con dispositivos hápticos y ópticos.

Objetivo
Evaluar la viabilidad del uso de tecnologías de realidad virtual para aplicaciones educativas y de inclusión.
